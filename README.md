### Skru Card Game

A console-based implementation of the **Skru** card game developed in C++. The game supports four players, special card abilities, score calculation, randomized deck shuffling, and an interactive command-line interface with colored terminal output.


### Overview
This project recreates the Skru card game using object-oriented and modular programming techniques. Players compete by managing their cards strategically while using special abilities to minimize their final score.
The project focuses on implementing game mechanics, player interactions, and card effects while maintaining a structured and reusable codebase.


### Features

- Four-player gameplay
- Randomized deck using Fisher–Yates Shuffle
- Score calculation
- Hidden cards system
- Multiple special card abilities
- SKRU announcement mechanic
- Turn-based gameplay
- Colored console interface
- Winner and loser determination
- Replay option


### Special Cards

- Basra
- Khod & Hat
- Khod Bas
- Ka3b Dayer
- Skru
- Skru Double
- +20
- -1
Each card performs its corresponding action during gameplay.
